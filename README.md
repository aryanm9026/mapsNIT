⚡ Getting Started

Run mapsNIT locally in just 3 steps 🚀

# 1. Clone the repository
git clone https://github.com/your-username/mapsNIT.git
cd mapsNIT

# 2. Install dependencies
npm install

# 3. Start the development server
npm run dev


Now open 👉 http://localhost:3000
